# 🍳 AI Recipe Generator

Full-stack AI-driven cooking platform.

## Quick Start

```bash
cp .env.example .env
# Fill in your values

# Backend
cd backend && npm install && npm run db:push && npm run dev

# Frontend (new terminal)
cd frontend && npm install && npm run dev
```
